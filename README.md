# Linux Mini Shell

A custom Linux shell implemented in C to understand Linux process management, command execution, and signal handling.

## Features

- Internal command execution
- External command execution
- Foreground process execution
- Background process execution
- Signal handling
- Process creation using fork()
- Command execution using execvp()

## Technologies Used

- C
- Linux
- POSIX APIs
- System Calls

## Concepts Covered

- Process Management
- fork()
- execvp()
- waitpid()
- Signals
- Shell Design

## Build

```bash
make
```

## Run

```bash
./a.out
```
